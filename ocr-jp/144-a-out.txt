Angel Notes.

鋼 の 大 地 [over count 1999.]

臨終 し た 星 。 死に 多 え た 惑星 。 生物 の 住め な い 世 界 。

現在 の 世界 の 名 称 。 正式 名 称 で は な く 、 荒 府 し た 大 地 に 生き ぎる ヒト 人 の 間 に 伝 わっ た 俗 若 。
鋼 の 大 地 の 記 名 が 示す と お り 、 現 在 の 大 陸 の 大 部 分 は ひび 割れ た 荒 野 で あり 、 据 色 と 白 過
と し た 愛 に 覆 わ れ て いる 。

食物 は 衣 つ こと は な く 、 大 気 は か つて の よう に 動物 に 適し た 物 で は な く な っ て いる 。

人 間 種 うに 高 う の な ら 、 役 底 し た 世紀 末 。 け れ ど 母体 で ある 星 が 死 記し て も 、 人 則竹 は
その 発 和 害し た 文明 技術 に よっ て 生き 長らえ た 。

か つて 人 る 々 が 因 い 描い た 星 の 終 わり で さえ 、 人 半 種 を 滅 記さ せる 事 は で き な か っ た 。

 

明白 種 [a-ragyj

星 の 資 源 を 使い ぎっ た 人 類 が 生み 出し た 次 世代 の 填 長 類 。

か つて の 量 に 現存 し て いた 各 和 生物 種 を モチ ー フ に し た も の 。 訪 廃し た 星 で も 生存 可能 で あ
る よう に 生 上 襲 を 大 きく 改良 ・ 強 化 さ れ た も の 。 その 委 統 掛 は 様々 で 、 大 きく 人 分類 し て 吾 竹 に
わた る 。 一 か ら 十 まで の 倍 で ある 亜 臣 は 単 一 種 で あり 、 郡 体 で は な い 。

中 に は 人 間 と し て の 遺伝 子 を る 含み 、 人 間 と 似 た 形態 の 亜 角 も 存在 する が 、 や は り 大 部 分 は
接 々 な 生命 竹 と 走 長 類 と が 融合 、 進 化し た も の で ある 。

人 間 種 rinen

鯛 の 大 地 に お ける 人 類 。 か つて の 人 則竹 が 、 そ の まま の 答 で この 世界 に 対 記 で きる よう に
進化 し た も の 。 正確 に は 彼ら も 亜 貿 に 含ま れる 。

現在 の 環境 に 対応 し て 生活 が で きる が 、 や は り 人 人間 と し て の 揚 能 を 上 回 る 能力 は な い 。
か つて の 文明 社会 を 再生 し よう と 国家 を 形成 し て いる 。 亜 由 吾 種 と は 不可 優 状態 。

大 戦 [Babel's Tale]

量 の 臨終 後 、 生 ぎ 延 び た 人 類 と 亜 百 種 の 識 に お きた 戦 巴 。

生き 残り を か けた 人 類 と 、 世 界 の 音 槍 を か けた 亜 前 と の 色 い 。

郡 体 で は な か っ た 亜 負 を 統一 し た 六 人 姉妹 の 前 に 人 類 は 敗北 李 前 まで お いこ まれ た 。 大 戦
末期 、 人 類 側 は 人 則竹 と 騎士 を 生み 出し 、 戦 い は 死 し た 時 を さら に 和 滅 させ る 大 戦へ と 発 太
する 。

大 戦 の 勝者 は いな い 。 瑞 勢力 の 戦い は 、 容 如 飛 来 し た 第 三 者 の 手 に よっ て どちら も 壊滅 す
前 に 追い 込ま れる 事 で 幕 を お ろ し た 。

馬 [ther Linerl]

人 間 種 の 中 で も 源 変 し た 環境 の 影響 を より 強く 受諾 し た 生命 種 。 府 侯 と 呼ば れる 持 殊 兵器
を 使用 する 。 旧 時 代 の 兵器 の 助力 な し で 亜 毅 と 対等 に 競い あえ る 攻 性 竹 。
現在 は 士 十 へ 人 が 登録 され て いる 。

魔 剣 [knight arms]

遇 士 が 持つ 武器 の 絵 特 。

この 世界 に 生ま れ た 人 間 種 は 、 す べから く ジ ン の 影響 を うけ る 。 生誕 の 際 に ジン を 多く 体
内 に 含ん で 誕生 し た 赤子 は 、 そ れ を 自ら の 張子 と し て 衣 て 上 げ 、 成 人 の さい に 外界 に カタ チ
と し て 産み 落と す 。 こ の 、 い まだ 未 解 析 の ジン で 形成 され た 兵器 は 様々 な 現象 を 誘 先 し 、 そ
の 影 響 は 二 グ に 兵器 と よ べ る レベ ル を も っ て いる 。

旗 記 を カタ チ に で きる 人 贈 は 数 池 な く 、 中 で も 実戦 に も ちい られ る ほど 強力 な 府 剣 を 持つ
も の を 騎士 と 呼ぶ 。 一 人 の 騎士 に つき 腐 語 は 一 振り と され る 。

ジン krain “Ether]

宇宙 神 。 星 と し て の 機能 を な くし た 時 に あぶ ふれ だ し た あら ぬる 有害 、 計 測 不 態 の 欄 子 の
絵 術 。 人 体 に 有害 で ある が 、 ご くま れ に 人 体 に 特異 な 変化 を も た ら す 事 か ら エ ー テ ル と も モ も 呂
ば れる 。

亜 臣 吾 種 、 人 間 種 、 遇 十 、 と も に この ジン に よっ て 生み 出さ れ た 新種 に すぎ な い 。 大 気 中
に 人 分散 す る ジン の エネ ルギー 安 損 率 は 圭 ま じ く 、 結 果 と し て か つて の 勤 星 上 で は あり えな か
っ た 戦記 理 論 が 打ち 出さ れ た 。

ジン を 体内 に 取り 込め る 亜 遇 、 ジ ン を 結晶 化 さ せ た 記 割 と の 戦 硬 に お いて 、 旧 時 代 の 兵器
は 全て が 無 価値 な 物 と な っ て し まっ た 。
